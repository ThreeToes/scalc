Building
========

Run the following

`gradle build`

Running Tests
=============

Run this
`gradle clean build test`

Running Application
===================

Run
`gradle run`

Alternatively, run 
`gradle clean fatJar`
and run scalc-all.jar from build/libs with 
`java -jar scalc-all.jar`
